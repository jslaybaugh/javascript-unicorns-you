Brief History of JavaScript

Created in 1995 by Brendan Eich

... in 10 days (he's wicked smart).

The Details

Brendan worked for Netscape.

Netscape was fighting with Microsoft over "the Internet."

Java (made by Sun Microsystems) was the new hotness.

Netscape viewed their browser as a lightweight OS with the JVM.

Netscape wanted easy interpreted language for the masses.

Called JavaScript to ride Java's popularity & scratch Sun's back.

JS had to 'look like Java' only less so, be Java's dumb kid brother or boy-hostage sidekick. Plus, I had to be done in ten days or something worse than JS would have happened.

- Brendan Eich (in 2010)

More Details

So, since Netscape 2.0 had JavaScript, IE 3.0 needed... JScript!

Well, flash was also kind of a big deal, so Adobe Macromedia decided they'd create ActionScript!

Some smart people (Brendan Eich again) decided we needed a standard for this new scripting language.

In 1996, the European Computer Manufacturers Association (ECMA) began standardizing the specification for what would be called ECMAScript.

ECMAScript was always an unwanted trade name that sounds like a skin disease.

- Brendan Eich (in 2012)

Oh. Sounds... Kinda crappy actually.

So, amidst all the politics and dumbing down, not to mention it was modeled after Java, JavaScript had many early hurdles.

But in spite of all that, what Brendan came up with in those 10 days has turned into something very powerful.

Dynamic, interpreted, flowing, procedural, prototypal, functional, extensible, scalable, asynchronous, event-driven...

JavaScript has taken on a life of its own and while it was once scorned, it is now one of the most popular languages.

Any application that can be written in JavaScript, will be written in JavaScript.

- Atwood's Law

Language Basics

  • Case-sensitive
  • White-space-agnostic
  • (Optional) semicolon statement termination
  • Comments: // or /* */
  • 4 main types: number, string, boolean, object
  • Control statements with parentheses and curly braces
  • ||, &&, !
  • =, ==, ===, !=, !==
  • Declare variables using var
  • There's a bunch more

function kindasort(arr)
{
    var temp = [];
    for (var i = 0; i < arr.length; i++)
    {
        if (temp.length <= 0)
        {
            temp.push(arr[i]);
        }
        else
        {
            for (var j = 0; j < temp.length; j++)
            {
                if (temp.length == 1) // 1 item array
                {
                    if (arr[i] < temp[j])
                    { 
                        // add to beginning of array
                        temp.unshift(arr[i]);
                    }
                    else
                    {
                        // add to end of array
                        temp.push(arr[i]); 
                    }
                }
                // if i'm at the last item in array
                else if (temp.length == j + 1)
                {
                    if (arr[i] < temp[j])
                    {
                        // add before current index
                        temp.splice(j, 0, arr[i]); 
                    }
                    else
                    {
                        // add after last item (which is this item)
                        temp.push(arr[i]); 
                    }
                }
                else
                {
                    if (arr[i] < temp[j])
                    {
                        // add before current index
                        temp.splice(j, 0, arr[i]); 
                    }
                    else
                    {
                        // add after this item, before next item
                        temp.splice(j + 1, 0, arr[i]); 
                    }
                }
            }
        }
    }

    return temp;
}

var newArray = kindasort([134, 4, 456, 23, 62, 0]);
